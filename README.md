# Test Data

This repository holds various test data, speicifcally for projects:

* [PerSeqPIPE](https://github.com/ktrachtova/perseqpipe)

## PerSeqPIPE Test Data

Data currently pushed to folder `/perseqpipe` are small datasets for different library preparation kits used by PerSeqPIPE for testings.

> [!IMPORTANT]
> Test data do not have to be manually downloaded to test the PerSeqPIPE. Test profiles of PerSeqPIPE pick the data automatically!
